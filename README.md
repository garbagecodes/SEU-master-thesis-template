## 本模板修改内容

- 在Windows下，使用AutoFakeBold来表示加粗宋体。

- 将图、表列表中的内容修改为图1-1、表1-1的形式，原模板中的列表只有1-1，没有“图”、“表”字样。

- 将原模板适用于双面打印的形式修改为单面，去掉了一些空白页。
